# 山易排盘

面向 Web 与后续原生 App 演进的四柱八字、岁运合参、易经求卦和传统命理学习工具。

## 当前能力

- 公历/农历日期与时间选择器，农历闰月、未知时辰与出生时间可信度
- 本地出生地点搜索选择，自动联动经度、纬度与标准时区，并保留自定义地点输入
- 经度、时区、夏令时、真太阳时和两种子时换日口径
- 四柱、藏干、十神、纳音、空亡、十二长生、神煞与干支关系
- 五行气势、格局、喜用、寒暖燥湿调候及分层改善方案
- 专业详批、专业细盘、大运/流年/流月/流日联动
- 易经三枚铜钱起卦、逐爻动画、本卦/变卦/动爻详解
- 本地全文古籍、现代译解、24 章 72 节课程、50 条术语、42 道分层练习与学习进度
- 今日复习台自动汇总未学课程、错题和古籍书签，实盘案例保留证据链、反证与常见误判
- 学堂支持简体中文与繁體中文本地切换，转换字典随应用打包、按需载入，语言偏好保存在本机
- 本机账号注册/登录、加密档案、数据导出/删除、隐私协议、内容边界和反馈队列
- PWA 安装与离线回访、联网状态提示、错误边界、可选匿名事件接口
- 键盘焦点、减少动态效果、弹窗焦点回收、档案删除确认等上线交互保护

## 数据边界

公开测试版默认不要求手机号。账号密码通过 PBKDF2-SHA-256 派生验证信息和 AES-GCM 密钥，密码本身不保存；命盘档案、出生资料、学习记录和反馈加密后保存在当前浏览器 `localStorage`，排盘也在浏览器本机完成。不同本机账号使用独立加密仓库，退出后需要重新输入密码解锁。游客模式不持久化个人记录。

这仍不是云账号系统，不支持跨设备同步或密码找回。清理浏览器数据、忘记密码或设备损坏都可能导致记录无法恢复，用户可先从“数据管理”导出 JSON 备份。接入正式后端前，不应在页面上宣称已经具备会员、支付、云备份或短信登录。

## 本地开发

```bash
npm ci
npm run dev
```

完整检查：

```bash
npm run check
npm run audit:release
```

其中包含 TypeScript 检查、排盘回归样例、账号加密与隔离测试和生产构建。测试不依赖额外测试框架，使用 Node 内置测试运行器。

## 可选运营接口

复制 `.env.example` 为本机环境文件并按需填写：

```bash
VITE_ANALYTICS_ENDPOINT=
VITE_FEEDBACK_ENDPOINT=
```

- 未配置统计接口时，即使用户同意匿名统计，事件也只保存在本机。
- 未配置反馈接口时，已登录用户的反馈进入账号加密仓库；游客反馈不会持久化，并明确告知用户尚未远程提交。
- 接口只能接收去标识化事件；不得把姓名、生辰、地点或完整报告发送到统计服务。

## 发布

推送 `main` 后，GitHub Actions 会执行类型检查、回归测试和 GitHub Pages 构建，再发布 `gh-pages` 分支。

正式发布前逐项执行 [RELEASE_CHECKLIST.md](./RELEASE_CHECKLIST.md)，至少确认账号边界、数据导出/删除、移动端关键路径、反馈接收端和回滚提交。

```bash
git push origin main
```

当前公开测试地址：<https://demonlion12.github.io/natural-paipan/>

正式商业运营仍需独立后端、真实身份认证、数据库备份、服务监控、内容审核、支付与退款流程，并将页面内协议交由专业人士复核。正式域名与备案配置暂不包含在本阶段。

## 计算口径

报告展示计算版本、原始输入、公历转换、有效排盘时刻、经度校正、均时差、夏令时、换日规则和时间可信度。当前回归样例覆盖：

- 已知四柱案例
- 农历转公历
- 真太阳时与夏令时校正
- 未知时辰与误差警告
- 午夜/子初换日差异

传统命理解读用于文化学习与结构化观察，不替代医学、心理、法律、财务等持证专业意见。
