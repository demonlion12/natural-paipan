# 山易排盘上线检查表

## 每次发布

- [ ] `npm ci`、`npm run check`、`npm run audit:release` 全部通过
- [ ] 游客进入、注册、登录、退出和错误密码提示可用
- [ ] 新建命盘、修改命盘、保存档案、打开档案、删除档案可用
- [ ] 公历/农历、未知时辰、真太阳时和地点匹配各抽测一例
- [ ] 流年、流月、流日可点击和横向滚动，流月为 12 个月，流日覆盖整月
- [ ] 起卦的自动模式、逐爻模式、阶段结果、本卦和变卦可用
- [ ] 学堂搜索、课程、练习、古籍章节、原文/导读切换可用
- [ ] 数据导出、清空账号内容和删除账号可用
- [ ] 手机 390px、平板 768px、桌面 1440px 无横向溢出和文字遮挡
- [ ] 断网后已缓存页面可回访，恢复联网后功能正常
- [ ] GitHub Actions 成功，线上版本号、脚本哈希与本次提交一致

## 正式运营前一次性完成

- [ ] 配置真实 `VITE_FEEDBACK_ENDPOINT`，验证反馈能进入有人处理的工单或邮箱
- [ ] 确定是否启用匿名统计；启用时配置 `VITE_ANALYTICS_ENDPOINT` 并检查事件不含生辰、地点和报告正文
- [ ] 用户协议、隐私说明、内容边界由法律与内容负责人复核并留存版本
- [ ] 建立内容勘误、投诉、未成年人和高风险表述处理流程
- [ ] 明确数据保留期、备份方式、恢复演练和账号迁移方案
- [ ] 若开始收费，补齐会员权益、订单、支付、发票、退款和客服流程
- [ ] 若需要跨设备账号，迁移到正式后端认证；本机账号不能宣传为云账号
- [ ] 保存上一稳定版本提交号，准备一键回滚并完成一次演练

## 当前静态版边界

- 账号和加密档案仅存在当前浏览器，无法跨设备同步，也无法找回密码。
- GitHub Pages 不适合承载正式会员、订单和敏感数据服务；这些能力需要独立后端与可配置安全响应头的托管环境。
- 命理解读属于传统文化学习参考，不得替代医疗、法律、心理和投资等专业服务。
